A quick over view of your SODA installation. You can see the number of resources provisioned, the number of backends, the dataflow quantitites etc.
You can register a new backend by clicking on the "+" in the center of the screen.
In the dialog presented enter a valid name and select the type of Backend you want to register.

 


Please note: Keep the following handy for a quick registration experience:

  1. Region
  2. Endpoint
  3. Bucket
  4. Access Key
  5. Secret Key

You may not need all the fields for all the backend types.
Currently you can register the following backend types: AWS S3, Huawei OBS, Azure Blob Storage, Ceph S3, GCP Storage, Fusionstore Object, IBM COS, YIG and AWS and Azure File Storage backends(new feature*).

Registered backends can be viewed by clicking on the cloud icons of the respective backends. The cloud in the center lists all other backends.
You can modify the Access Key and Secret Key of the registered backend and delete it completely.
